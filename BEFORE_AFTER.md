```

```

## 💡 핵심 변화

### 🔴 Before: "추천이 맞을까?"
- 일반적인 추천만 봄
- 자신의 패턴과 연결 못 함
- "이거 나한테 필요한 건가?" 의심

### 🟢 After: "아, 내 패턴을 알고 추천했구나"
- **구체적인 데이터** (지난 7일, 오전 10-11시, 5회)
- **명확한 연결** ("당신의 이미지 처리 작업에 맞춤")
- **신뢰도 상승** ("이건 내 상황 기반이니까 써봐야겠다")

## 🎯 최종 효과

```
시간 절감 ⏱️
├─ 스킬 찾기: 40분 → 0분 (알림으로 옴)

├─ 설치율: 40% → 95%
└─ 실제 사용: 2-3개 → 4-5개

성장 가시화 📊
├─ "내 패턴이 뭐지?" → "아, 이런 일을 하구나"
├─ "효과 있었나?" → "_scout-recommendations.md로 추적"
└─ "다음 뭐하지?" → "지난주 데이터 기반 계획"
```
